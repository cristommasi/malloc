

• Manage malloc debug environment variables. You can imitate those from the system
malloc or invent your own {

    MALLOC_CHECK_         0-3
    0 = silent, ignore errors
    1 = print error message to stderr but continue
    2 = abort() immediately on any corruption
    3 = print + abort



    MALLOC_PERTURB_       0-1
    1 = on alloc: fill with custom poison byte


    MALLOC_VERBOSE        0-1
    1 = ft_malloc(32)  -> 0x7ffff7fb6120
    1 = ft_free(0x7ffff7fb6120)
    1 = ft_realloc(0x7ffff7fb6120, 64) -> 0x7ffff7fb6200


    MALLOC_HIST_SIZE      0-n
    How many past alloc/free events to keep in a ring buffer for the extended dump.


    MALLOC_MMAP_THRESHOLD 1025      size to go directly to mmap
    Size before using mmap directly LARGE_CHUNK_MIN
}
